# 📚 Chào mừng đến với Notepad

Đây là trang notepad **readonly**, chạy hoàn toàn trên GitHub Pages mà không cần backend.

## ✨ Tính năng nổi bật
- **Giao diện Light Mode**: Hiện đại, sạch sẽ, dễ đọc.
- **Hỗ trợ Markdown**: Viết ghi chú nhanh chóng với định dạng chuẩn.
- **Hỗ trợ Mermaid**: Vẽ sơ đồ, flowchart trực tiếp trong markdown.
- **Tìm kiếm tức thì**: Lọc note ngay trên trang chủ.

> 💡 **Mẹo:** Bạn chỉ cần thêm file `.md` vào thư mục `notes` và cập nhật `notes.json` là hệ thống sẽ tự động nhận diện.
